## ğŸ”§ Setup Instructions

### For GitHub Pages Deployment:

1. **Create Repository**:
   ```bash
   # Create new repository on GitHub named 'fengzhang-website'
   ```

2. **Upload Files**:
   ```bash
   git clone https://github.com/YOUR-USERNAME/fengzhang-website.git
   cd fengzhang-website
   # Copy all files from this folder to the repository
   git add .
   git commit -m "Initial website setup"
   git push origin main
   ```

3. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Save settings

4. **Add Custom Domain** (optional):
   - Add CNAME file with your domain
   - Configure DNS settings

### For Local Development:

1. **Clone Repository**:
   ```bash
   git clone https://github.com/YOUR-USERNAME/fengzhang-website.git
   cd fengzhang-website
   ```

2. **Serve Locally**:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using PHP
   php -S localhost:8000
   
   # Using Node.js (if http-server installed)
   npx http-server
   ```

3. **View Website**:
   Open `http://localhost:8000` in your browser
